# 🌍 Global Control Europe Leaderboard

A premium, interactive leaderboard for tracking the absolute best **Roblox Control Europe** players.

Designed with a sleek, space-inspired dark theme and fully responsive architecture, this platform tracks the top 1v1 Skill and Event Wins players within the CE community. 

## ✨ Features

- **🏆 Centralized Standings:** Track top players across `Overall Skill` and `Event Wins` seamlessly on a single page.
- **🎨 Premium UI/UX:** Built with modern web standards, featuring glassmorphism elements, custom CSS animations, and highly polished layouts.
- **🛡️ Integrated Admin Panel:** A secure, intuitive frontend interface (`/admin.html`) allowing authorized users to edit, add, or sort players dynamically.
- **📱 Fully Responsive:** Clean viewing experience and a fully functional custom navigation menu on both desktop and mobile devices.
- **📜 Clan Lore & Updates:** Built-in pages for community lore and patch notes tracking.

## ⚙️ How it Works

The entire project operates as a fast, statically hosted web application. Player data is safely maintained in `data.js` and loaded dynamically.

**Admin Workflow:**
1. Navigate to `admin.html` (or click the floating badge on the main site).
2. Enter the administrator passcode.
3. Manage players using the drag-and-drop dashboard, update wins, and clan standings.
4. Click **Export**, copy the updated payload, and paste it into `data.js` via a GitHub commit to apply changes live.
